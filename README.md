## About The Project

NOTE: This project is still under active development. Watch out for updates!

My old website was basically just copied from an online tutorial that I found, with some minor edits. I taught myself React and Typescript over the summer, so I wanted to revamp my old website to add some more personal flair and to show off my new skills. The core idea here is minimalism.
